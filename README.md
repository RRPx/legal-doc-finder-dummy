# legal-doc-finder-dummy
A dummy legal site finder

# How to run:
1. Clone - git clone git@github.com:RRPx/legal-doc-finder-dummy.git
2. Make sure you've opened Docker (I used Docker Desktop for Windows)
3. cd legal-doc-finder-dummy
4. docker compose build --no-cache
5. docker compose up (it should enable two ports on your machine)
6. Please go to the frontend link (it will be shown to the terminal)
